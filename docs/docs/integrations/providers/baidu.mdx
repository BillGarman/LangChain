# Baidu 

>[Baidu](https://www.baidu.com/) is the most used widely search engine in China. At the meantime, Baidu is also an influential AI Technology company which has create `ERNIE-Bot`, `Qianfan`, `Paddle` and so on.

## Qianfan Large Model Platform

[Baidu Intelligent Cloud's Qianfan Large Model Platform](https://cloud.baidu.com/doc/WENXINWORKSHOP/index.html) (hereinafter referred to as Qianfan or Qianfan Large Model Platform) is a one-stop platform for the development and service operation of large models tailored 

for enterprise developers. Qianfan not only provides foundational models like ERNIE-Bot and third-party open-source large models but also offers a variety of AI development tools and a complete 

development environment, enabling customers to easily use and develop applications with large models.
