# dstack

>[dstack](https://dstack.ai/) is an open-source tool for orchestrating GPU across multiple cloud providers.

## Installation and Setup

Transformers library is used for chat templating. To install it, run:

```bash
pip install transformers
```

## LLMs

### dstack

See a [usage example](/docs/integrations/llms/dstack).

```python
from langchain.llms import Dstack
```
