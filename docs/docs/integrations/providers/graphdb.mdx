# GraphDB

>[GraphDB](https://graphdb.ontotext.com/) is a graph database and knowledge discovery tool compliant with RDF and SPARQL.

## Dependencies

Install the [rdflib](https://github.com/RDFLib/rdflib) package with
```bash
pip install rdflib==6.3.2
```

## Graph QA Chain

Connect your GraphDB Database with a chat model to get insights on your data.

See the notebook example [here](/docs/use_cases/graph/graph_graphdb_qa).

```python
from langchain_community.graphs import GraphDBGraph
from langchain.chains import GraphDBQAChain
```
