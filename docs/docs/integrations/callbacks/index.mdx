# Callbacks


LangChain provides a **callbacks system** that allows you to hook into the various stages 
of your LLM application. This is useful for logging, monitoring, streaming, and other tasks.

Callback Handler is an object that implement the `CallbackHandler` interface, 
which has methods for each event that can be subscribed to. 

See more in [Docs](/docs/modules/callbacks/).
