How To Guides
====================================

There are a lot of different document loaders that LangChain supports. Below are how-to guides for working with them

`File Loader <./examples/unstructured_file.ipynb>`_: A walkthrough of how to use Unstructured to load files of arbitrary types (pdfs, txt, ipynb, etc).

`Directory Loader <./examples/directory_loader.ipynb>`_: A walkthrough of how to use Unstructured load files from a given directory.

`Notion <./examples/notion.ipynb>`_: A walkthrough of how to load data for an arbitrary Notion DB.

`ReadTheDocs <./examples/readthedocs_documentation.ipynb>`_: A walkthrough of how to load data for documentation generated by ReadTheDocs.

`ipynb <./examples/ipynb.ipynb>`_: A walkthrough of how to load data from an ipynb file.

`PDF <./examples/pdf.ipynb>`_: A walkthrough of how to load data from a PDF file.

`PowerPoint <./examples/powerpoint.ipynb>`_: A walkthrough of how to load data from a powerpoint file.

`Email <./examples/email.ipynb>`_: A walkthrough of how to load data from an email (`.eml`) file.

`GoogleDrive <./examples/googledrive.ipynb>`_: A walkthrough of how to load data from Google drive.

`Microsoft Word <./examples/microsoft_word.ipynb>`_: A walkthrough of how to load data from Microsoft Word files.

`Obsidian <./examples/obsidian.ipynb>`_: A walkthrough of how to load data from an Obsidian file dump.

`Roam <./examples/roam.ipynb>`_: A walkthrough of how to load data from a Roam file export.

`EverNote <./examples/evernote.ipynb>`_: A walkthrough of how to load data from a EverNote (`.enex`) file.

`YouTube <./examples/youtube.ipynb>`_: A walkthrough of how to load the transcript from a YouTube video.

`Hacker News <./examples/hn.ipynb>`_: A walkthrough of how to load a Hacker News page.

`GitBook <./examples/gitbook.ipynb>`_: A walkthrough of how to load a GitBook page.

`s3 File <./examples/s3_file.ipynb>`_: A walkthrough of how to load a file from s3.

`s3 Directory <./examples/s3_directory.ipynb>`_: A walkthrough of how to load all files in a directory from s3.

`GCS File <./examples/gcs_file.ipynb>`_: A walkthrough of how to load a file from Google Cloud Storage (GCS).

`GCS Directory <./examples/gcs_directory.ipynb>`_: A walkthrough of how to load all files in a directory from Google Cloud Storage (GCS).

`Web Base <./examples/web_base.ipynb>`_: A walkthrough of how to load all text data from webpages.

`IMSDb <./examples/imsdb.ipynb>`_: A walkthrough of how to load all text data from IMSDb webpage.

`AZLyrics <./examples/azlyrics.ipynb>`_: A walkthrough of how to load all text data from AZLyrics webpage.

`College Confidential <./examples/college_confidential.ipynb>`_: A walkthrough of how to load all text data from College Confidential webpage.

`Gutenberg <./examples/gutenberg.ipynb>`_: A walkthrough of how to load data from a Gutenberg ebook text.

`Airbyte Json <./examples/airbyte_json.ipynb>`_: A walkthrough of how to load data from a local Airbyte JSON file.

`Online PDF <./examples/online_pdf.ipynb>`_: A walkthrough of how to load data from an online PDF.

`CoNLL-U <./examples/CoNLL-U.ipynb>`_: A walkthrough of how to load data from a ConLL-U file.

`iFixit <./examples/ifixit.ipynb>`_: A walkthrough of how to search and load data like guides, technical Q&A's, and device wikis from iFixit.com
