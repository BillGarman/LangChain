# AI21 Labs

>[AI21 Studio](https://docs.ai21.com/) provides API access to `Jurassic-2` large language models.
>See the [AI21 models](https://docs.ai21.com/docs/jurassic-2-models).

## Installation and Setup

Get an AI21 api key and set it as an environment variable (`AI21_API_KEY`). 
See [instructions](https://studio.ai21.com/account/account).

```bash
pip install ai21
```


## LLMs

```python
from langchain.llms import AI21
```
